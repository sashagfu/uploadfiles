## Quick Start

Before installing, [download and install Node.js](https://nodejs.org/en/download/).
Node.js 0.10 or higher is required.

  Go to the folder:

```bash
$ express /tmp/foo && cd /uploadfiles
```

  Install dependencies:

```bash
$ npm install
```

  Start the server:

```bash
$ node app.js
```
